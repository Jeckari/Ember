Ember is a simple little note organization tool.
It is provided as-is, under a modified BSD License (see license.txt). 

Ember is open source, with the main repository being https://github.com/Jeckari/Ember
If you fix bugs or add features, please submit a pull request. 

To set up, there is one file you'll need to look at, cgi-bin/config.php. 
Set the variables there to reflect your system.
